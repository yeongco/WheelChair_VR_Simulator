# WheelChair_VR_Simulator
본 프로젝트 주제는 VR 지하철역 휠체어 이동 체험입니다.
현실적인 체험을 기반으로 장애인 이동 환경에 대한 문제의식을 높이고, 개선의 필요성을 느낄 수 있도록 돕는 것이 본 프로젝트의 핵심 목표입니다.
